# SEC Filings Guide

A single-page reference for SEC filings. Content is drawn from DFIN and SEC resources.

## View the site

Open `index.html` in any browser.

## Project files

| File        | Purpose                                                |
|------------|--------------------------------------------------------|
| `index.html` | Overview, key takeaways, **What/Who/When table**, form details, Form 10 deep dive, how to look up filings |
| `styles.css` | Layout and typography                                  |
| `script.js`  | Filter form cards by category                          |
| `content.md` | Source content notes                                   |

## Features

- **Key takeaways** — 10-K, 10-Q, 8-K, proxy, EDGAR
- **Enhanced table** — Every form with **What** (purpose), **Who** (filer), **When** (deadline/trigger)
- **Form details** — Each filing explained with What, Who, When + investor tips
- **Form 10 deep dive** — Components, compliance (from DFIN)
- **How to look up filings** — EDGAR, public access, bottom line
